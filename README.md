## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

#prisma studio (port:5555)
$ px prisma studio

#swagger (port:3001/swagger)
$ npm run start
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```
